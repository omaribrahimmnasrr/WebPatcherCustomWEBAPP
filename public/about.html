What is Mini Web App?

This is a simple web application built to demonstrate the basics of web development using HTML, CSS, JavaScript, and Node.js with Express framework.

The application features a clean, modern design with responsive layout, multiple pages, and API endpoints for dynamic content.

Technologies Used:

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Node.js, Express.js
  • Features: Static file serving, API routes, Form handling
